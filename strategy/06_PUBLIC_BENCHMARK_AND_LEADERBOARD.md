# AI PR Verifiability Benchmark

Rank coding agents on evidence quality, not aesthetics:
- valid behavioral claims found;
- regression witnesses;
- vacuous-test rate;
- valid counterfactual kill rate;
- replay success;
- evidence-boundary completeness;
- time/cost and inconclusive rate.

Use hidden sibling fixtures and rotating inputs. Checker runs outside the code-generating agent. Publish methodology, confidence intervals and failures. Never let model vendors pay for rank. Begin with controlled fixture PRs; add real OSS PRs only after permission/responsible disclosure.
